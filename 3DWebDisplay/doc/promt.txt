Bạn là một Senior Frontend Engineer chuyên về WebGL và Three.js, có kinh nghiệm xây dựng 3D portfolio website hiệu năng cao chạy hoàn toàn trên GitHub Pages, chỉ sử dụng HTML, CSS, JavaScript (ES6+), không backend.

Hãy tạo một website với các yêu cầu sau:
1. Tổng thể thiết kế
    Phong cách hiện đại, tối giản, high-tech, tập trung 100% vào việc trưng bày 3D model
    UI rõ ràng, không rối, không dùng framework nặng (React/Vue không bắt buộc)
    Ưu tiên hiệu năng, khả năng mở rộng và khả năng tự chỉnh sửa code

2. Trưng bày 3D model
    Sử dụng Three.js + WebGL
    Load model định dạng glTF / GLB
    Hỗ trợ Draco compression (giải nén phía client)
    Model có texture đã bake (albedo, normal, roughness, metallic…)
    Tự động:
        Center model về gốc tọa độ
        Scale model phù hợp viewport
        Đặt camera target = tâm model

3. Tối ưu model
    Có pipeline JS để:
        Lazy load model
        Chỉ load khi người dùng chọn
        Tách logic load / render / dispose rõ ràng
        Hỗ trợ unload model khi không cần thiết để giải phóng bộ nhớ

4. Hệ thống chọn model

Có một model selector panel
    Mỗi model là một item:
        Hover sẽ kéo nền sang bên cạnh
        Hiện tên model overlay trên nền
        Animation mượt, dùng CSS + GSAP (nếu cần)
    Code phải cực kỳ đơn giản để thêm model mới, ví dụ: 
        Chỉ cần thêm object vào một mảng JS:
            tên
            đường dẫn model
            thumbnail

5. Trang mặc định (Home / About)
    Khi chưa chọn model:
        Hiển thị thông tin chung của tôi
        Tên
        Mô tả ngắn
        Link mạng xã hội
    Icon mạng xã hội:
        Hình tròn
        Có tên bên cạnh
    Khi mở model 3D:
        Khối thông tin này co gọn lại, chuyển sang layout phụ, không che model

6. Điều khiển camera
    Camera:
        Chỉ xoay 2 trục (ngang & dọc)
        Zoom in / zoom out
        Không cho roll
    Không cho pan tự do
    Điều khiển mượt, có damping

7. Hiệu ứng & chuyển động (Bonus)
    UI có cảm giác “chìm vào không gian 3D”
    Nút bấm và panel có:
        Depth
        Parallax nhẹ
        Hover tạo cảm giác 3D
    Phong cách tham chiếu:
        UI sci-fi
        Tương tự cảm giác nút bấm trong game Arknights: Endfield

8. Code & cấu trúc
    Tách file rõ ràng:
        index.html
        style.css
        main.js
        viewer.js (Three.js logic)
        models.js (danh sách model)
    Code phải:
        Dễ đọc
        Có comment ngắn gọn
        Không over-engineering

9. Yêu cầu đầu ra
    Sinh code hoàn chỉnh
    Chạy được ngay trên GitHub Pages
    Không dùng backend
    Không dùng thư viện nặng không cần thiết
    Toàn bộ giải thích và comment bằng tiếng Việt
    Tên biến, class, function bằng tiếng Anh
